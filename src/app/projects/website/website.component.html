Ontdekken

Als laatste activiteit heb ik gekozen om een eigen website te ontwikkelen die later kan dienen als visitekaartje bij sollicitatiegesprekken. Omdat Angular mij bijzonder heeft geboeid in het eerste trimester van het schooljaar en omdat mijn stage voornamelijk bestond uit programmeren in Angular wou ik mij graag verdiepen in wat dit framework nog meer te bieden heeft.

Ik heb gebruik gemaakt van Angular Material, een UI-componentenbibliotheek voor het bouwen van aantrekkelijke, consistente en functionele webpagina's. In de eerste plaats heb ik de documentatie van Angular Material doorgenomen en gekeken welke componenten ik kon gebruiken voor mijn website.

Daarnaast heb ik ook gebruik gemaakt van Tailwindcss voor een custom design te creëren. In tegenstelling tot Bootstrap komt Tailwindcss niet met voorgedefinieerde componenten maar voorziet het een set van CSS helper klasses. Deze klasses kunnen eenvoudig en snel gebruikt worden doorheen het project.

Mijn doel bij het ontwikkelen van deze website is om zoveel mogelijk kennis rond Angular, Angular Material en Tailwindcss op te doen voornamelijk omdat ik deze kant uit wil in mijn volgende job. Ik heb bij het maken extra aandacht besteed om de website zo gebruiksvriendelijk en intuïtief mogelijk te maken. Wat de layout betreft wou ik zoveel mogelijk verschillende klasses van Tailwind uitproberen. Ik wist op voorhand dat ik nooit hetzelfde niveau van website zou kunnen bereiken dan wanneer ik deze website zou maken met behulp van een content management systeem maar het hoofddoel hier was niet om de mooiste website te maken maar eerder zoveel mogelijk kennis rond bovenstaande technologieën op te doen.

Uitvoeren

Eens ik door alle documentatie ben gegaan was het tijd om aan de slag te gaan. Ik ben begonnen met het uitdenken van hoe de website er structureel moest uitzien. Hiermee bedoel ik meer bepaald het opdelen van de website in verschillende modules en de routering die hierbij hoort. Een module is een soort van container voor samenhangende code die de applicatie in kleinere onderdelen verdeelt. Modules zorgen voor een betere performantie doordat ze lazy loading mogelijk maken waardoor een module enkel laadt wanneer deze nodig is.
